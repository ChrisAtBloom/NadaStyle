NadaStyle
=============

A color schemefor Intellij

![My image](adanperez.github.com/NadaStyle/images/example.png)


Supports
-------

The following languages and features are supported at this time.

* Java
* Groovy
* HTML
* CSS
* Python
* LESS
* Javascript
* HandleBars
* SQL
* JSP
* XML
* Regex
* Properties
* Manifest

Installation
-----------
    Place file in:
    ~/Library/Preferences/IntelliJIdea12/colors
