NadaStyle
=============

Color Schemes for Intellij

Supports
-------

The following languages and features are supported at this time.

* Java
* Groovy
* HTML
* CSS
* Python
* LESS
* Javascript
* HandleBars
* SQL
* JSP
* XML
* Regex
* Properties
* Manifest

Installation
-----------
    Place file in:
    ~/Library/Preferences/IntelliJIdea12/colors
